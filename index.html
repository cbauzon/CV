About Me

I have spent most of my life in the Bay Area, where I was grew up surrounded by so much tech. When I was in middle school, I remember playing around with the command line on this very old Windows 7 computer. I was able to follow YouTube tutorials on how to change the command line fonts and colors, and of course, I had to change it to the bright green color to make it look like I was a "cool hacker". From then on, I became fascinated with what computers, and technology in general, can do, which eventually lead me to pursue a degree in electrical engineering.

Fast-forward to December 2022, I graduated with a B.S. in Electrical Engineering from California Polytechnic State University, San Luis Obispo. Through my years at Cal Poly, I have learned a lot about the intricacies of how computers and electronics function on the hardware level. And whenever I can, I try to improve my understanding of the software side of things on my own. I have experience scripting with Python to create small terminal-based apps and with C to program firmware drivers for simple embedded systems. Lately, I've been focusing on learning full-stack technologies like HTML, CSS, JS and some of its frameworks (jQuery, Node.js, Express.js), and both SQL and NO-SQL databases.

On my free time, I like to play video games; take photos of nature and cars; and continue improving my coding skills by learning new technologies. Check out the carousel below to see some of the photos I have taken.

Simon Game JS

Libraries used: jQuery

This is a browser-based rendition of the Simon Game. A random sequence of of blocks are selected, and the player has to remember the order to proceed to the next level. Each subsequent level will add another block in sequence, and the game gets progressively harder and harder.

What I Learned:

This game helped me practice adding more JavaScript logic to my website. It also helped me get practice with using jQuery to make using the DOM much more simpler and less repititive.

To-Do List JS

Libraries used: ejs, Node.js, Express.js

A simple to-do list that allows you to create and delete tasks using the plus button and the checkbox respectively. The tasks are stored in a MongoDB database, so they are persistent even after server restarts.

What I Learned:

This project taught helped me practice creating dynamic websites. More specifically, I learned how to interface a front-facing website with a server using JavaScript. I created, read, updated, and deleted data on a local MongoDB server with the help of Node.js and Express.js.
